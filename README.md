# word-knowledge-base

## 项目简介

word-knowledge-base 是一个基于 Obsidian 的英语单词知识库项目，用于系统化地管理和学习英语词汇。

## 目录结构说明

- `.obsidian/`: Obsidian项目配置目录，包含插件与样式文件
- `README.md`: 本文件

## 安装与部署

1. 安装 [Obsidian](https://obsidian.md/)
2. 将本项目克隆到 Obsidian 的 `vaults` 目录下
3. 启动 Obsidian，打开本项目

## 使用说明

本知识库用于记录和管理英语单词学习内容，支持：
- 单词记忆卡片
- 词汇分类整理
- 学习进度追踪
